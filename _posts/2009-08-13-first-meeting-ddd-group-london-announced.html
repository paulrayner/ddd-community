Gojko Adzic
has been kind enough to volunteer to present a talk entitled "DDD and
relational databases: the value object dilemma"

Value objects
are one of the corner-stones of domain-driven design but teams often
struggle to use this pattern efficiently when they need to be stored to
relational databases.

In this presentation, Gojko will explore
several approaches to persisting value objects, presents best practices
and talks about common pitfalls for value object persistence.

The meetup will take place on Monday, September 28 at 6:30 p.m.

We are happy to announce that a new version of DDD Sample Application is released. [read full story on Yahoo! Groups]

DDDSample is a working sample application based on the cargo shipping domain, built in Java and a few popular frameworks like Spring, Hibernate, CXF and ActiveMQ. It is a collaborative effort between Domain Language and the Swedish consulting company Citerus.
